# 个人任务管理工具

## 项目背景
开发一个本地运行的个人任务管理 Web 应用，单用户使用，数据存储在本地 JSON 文件中，不需要数据库。

## 技术栈
- 后端：Node.js + Express
- 前端：纯 HTML + CSS + JavaScript（不用框架，单文件即可）
- 数据存储：本地 JSON 文件（data/tasks.json）
- 端口：5500

## 功能需求

### 核心功能
1. **任务 CRUD**
   - 创建任务：标题、描述、优先级（高/中/低）、截止日期
   - 编辑任务的所有字段
   - 删除任务（带确认）
   - 标记任务完成/未完成

2. **任务列表**
   - 按状态筛选：全部 / 进行中 / 已完成
   - 按优先级筛选：全部 / 高 / 中 / 低
   - 按创建时间排序
   - 搜索任务标题

3. **数据统计**
   - 页面顶部展示：总任务数、已完成数、进行中数、今日到期数
   - 用简单的数字卡片展示

### 页面结构
- 单页面应用
- 顶部：标题栏 + 统计卡片
- 中间：筛选栏 + 搜索框 + 新建按钮
- 主体：任务卡片列表
- 每个任务卡片显示：标题、描述摘要、优先级标签、截止日期、完成状态复选框、编辑和删除按钮
- 新建和编辑用弹窗表单

### 样式要求
- 深色主题
- 优先级用颜色区分：高=红色、中=黄色、低=蓝色
- 已完成的任务显示删除线和降低透明度
- 响应式布局，手机也能用
- 页面打开时卡片有淡入动画

## API 设计
- GET    /api/tasks          获取所有任务
- POST   /api/tasks          创建任务
- PUT    /api/tasks/:id      更新任务
- DELETE /api/tasks/:id      删除任务

## 验收标准
- npm install && node server.js 可以直接启动
- 浏览器打开 http://localhost:5500 能正常使用
- 所有 CRUD 功能正常工作
- 页面刷新后数据不丢失
- 手机浏览器访问布局不乱
- 有 README.md 说明启动方式
