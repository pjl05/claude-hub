# 我的个人设定

## 身份
我是 [你的名字]，一个全栈开发者，日常工作是项目快速开发和交付。

## 技术偏好
- 前端：React 18 + TypeScript + Tailwind CSS（默认方案）
- 后端：Node.js + Express + Prisma（默认方案）
- 数据库：PostgreSQL（正式项目）/ SQLite（小工具）
- 包管理器：pnpm
- 代码风格：简洁实用，不过度设计

## 工作习惯
- 中文注释，技术术语保留英文
- 变量命名 camelCase，组件命名 PascalCase
- 优先 async/await
- 每个函数不超过 50 行
- 写代码的同时写文档
- 先确保功能正确，再优化性能
- 所有 API 必须有错误处理

## 项目规范
- 每个项目必须有 README.md，包含：项目说明、启动方式、目录结构
- 环境变量用 .env 文件，提供 .env.example
- 重要的配置项要有注释说明

## 沟通偏好
- 用中文交流
- 技术术语保留英文原文
- 问题描述要具体，附带代码示例
- 不要说废话，直奔主题
